Hướng dẫn nhanh để dùng và triển khai trang 'Gửi đến Bảo Vy yêu dấu'

Files included:
- index.html : Trang chính (HTML/CSS/JS đã nối sẵn)
- style.css  : Styles
- script.js  : JavaScript xử lý hiệu ứng, nhạc, và lời chúc
- README.txt : Tệp hướng dẫn này

Lưu ý về nhạc nền:
- Mình đã đặt sẵn YouTube embed cho "Perfect (Ed Sheeran)" trong script.js. Nếu bạn muốn dùng bài hát khác, hãy thay giá trị 'youtubeVideoUrl' trong script.js bằng đường dẫn embed của video YouTube bạn muốn (ví dụ: https://www.youtube.com/embed/VIDEO_ID?autoplay=1&rel=0).
- Tôn trọng bản quyền: đảm bảo bạn có quyền phát nhạc (nếu là nhạc có bản quyền, trang có thể phát qua YouTube embed; nếu muốn nhúng file mp3, hãy đảm bảo bạn có quyền sử dụng).

Cách chạy ngay trên máy của bạn:
1. Giải nén zip.
2. Mở file index.html bằng trình duyệt (kéo thả file vào cửa sổ trình duyệt).

Cách triển khai miễn phí nhanh trên Vercel:
1. Tạo tài khoản tại https://vercel.com (nếu chưa có).
2. Tạo một repository mới trên GitHub và đẩy 3 file (index.html, style.css, script.js) vào repository đó.
3. Trên Vercel, nhấn "New Project" -> Import từ GitHub -> chọn repo -> Deploy.
4. Sau khi deploy xong, bạn sẽ có domain dạng https://<tên-project>.vercel.app .
   - Muốn tên chính xác https://tienyeuvy.vercel.app bạn cần đặt tên project là 'tienyeuvy' (nếu tên đó chưa bị người khác dùng).

Nếu bạn muốn, mình có thể:
- Giúp tạo file ZIP này thành link tải trực tiếp (mình đã tạo ở server của bạn trong bước này) — tải xuống file: /mnt/data/tienyeuvy_site.zip
- Hoặc hướng dẫn chi tiết từng bước đẩy lên GitHub & Vercel.

Chúc bạn tạo được món quà thật ấm áp cho Bảo Vy!
